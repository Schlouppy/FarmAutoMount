Farm Auto Mount
===============

Automatically remounts after gathering herbs, ore, or lumber.
If you can gather while mounted, the addon won't try to remount.
After the 1.1.5 update, FarmAutomount could be a bit to chatty, by default it's now silent, but for those who already had the addon don't hesitate to manually shut it down : /fam silent 

Commands:
  /fam mount <name>      Set mount for this character
  /fam account <name>    Set mount for all characters
  /fam delay <seconds>   Set delay before mounting (default: 0.5)
  /fam enable            Enable auto-mount
  /fam disable           Disable auto-mount
  /fam reset             Reset character mount
  /fam reset account     Reset account mount
  /fam silent            Toggle error notifications
  /fam debug             Toggle debug mode

Tip: Shift+click a mount in your journal to paste its name.
