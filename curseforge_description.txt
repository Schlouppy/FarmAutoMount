# 🌿 Farm Auto Mount

Automatically resummons your mount right after gathering any resource — herbs, ore, lumber, or certain containers (chests, bone piles…). No more fumbling with keybinds between each node.

If you can gather without dismounting (e.g. mounted lumber harvesting), the addon detects it and stays out of the way.

> ⚠️ Even with mounted gathering, mobs can still knock you off — and chests/containers always require dismounting. So the addon stays useful regardless!

## ✨ Features

*   🐴 **Auto-remount** — Mounts automatically after looting a gathering node
*   📦 **Container support** — Also remounts after looting chests, bone piles, and other containers
*   🪵 **Mounted gathering** — Won't interfere if you can gather without dismounting (but mobs will still dismount you — and we've got you covered!)
*   🎯 **Custom mount** — Per-character or account-wide mount setting
*   ⏱️ **Configurable delay** — Adjust the delay before mounting (default: 0.5s)
*   🐻 **Druid-friendly** — Automatically skips druids (Travel Form is better!)
*   🔇 **Silent by default** — Error notifications disabled by default (`/fam silent` to toggle)
*   🌍 **10 languages** — EN, FR, DE, ES, IT, PT, RU, KO, ZH-CN, ZH-TW

## 📝 Commands

*   `/fam mount <name>` — Set mount for this character
*   `/fam account <name>` — Set mount for all characters
*   `/fam reset` — Reset character mount (use account setting)
*   `/fam reset account` — Reset account mount (use favourite)
*   `/fam enable / disable` — Toggle the addon
*   `/fam delay <seconds>` — Adjust the remount delay
*   `/fam silent` — Toggle error notifications
*   `/fam debug` — Toggle debug mode

💡 **Tip:** Shift+click a mount in your journal to paste its name into chat!

## ❓ FAQ

**I can gather lumber while mounted, will the addon cause issues?**
No! The addon detects if you're already mounted and won't try to remount.

**Why doesn't it work for my Druid?**
Druids are automatically skipped. Travel Form is instant and better than any mount.

**I don't want error messages in chat.**
Error notifications are silent by default since v1.1.5. Use `/fam silent` to toggle them on.
